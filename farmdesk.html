❓ FarmDesk Help

Quick guide to get started and use every feature.

🚀 Getting Started

1. Go to Settings and enter your farmOS server URL and login — or enable Standalone Mode if you have no server.

2. Go to Animals and add your animals. If connected to farmOS, tap FarmOS Sync → Fetch to pull them automatically.

3. Set your Active User in Settings so every log is stamped with your name.

4. Start logging! Tap any green button on the home screen to record a health event.

📋 Log Types

⚖️ Weight — Record an animal's weight in lbs or kg.

🌡️ Vitals — Temperature, heart rate, respiratory rate.

💉 Med — Medication name, dose, route, and lot number.

✂️ Procedure — Any procedure performed on an animal.

🐾 Hoof — Hoof trimming and condition notes.

👁️ Observe — General observation or health note.

🐐🍼 Birth — Record a birth event with dam, sire, and offspring.

📝 Activity — General farm activity log.

Weight + Vitals — Record both in one flow.

Weight + Vitals + Med — Record all three in one flow.

📦 Stock Module

Track feed, medical supplies, and lab items.

+ button — Add a new inventory item.

+ / − buttons — Adjust quantity up or down. You'll be prompted for amount and notes.

Tap item name — View full details and history.

History tab — Full log of all stock adjustments.

Red bar — Item is at zero or below. Restock needed.

📅 Scheduler

Schedule future health tasks for animals. Scheduled tasks appear on the home screen and in History.

Tap a scheduled task on the home screen to open the log form pre-filled and mark it complete.

📤 Data / Export

Export XML — Full backup of all logs, animals, and stock. Use the same filename every time to overwrite the previous backup.

Import XML — Restore from a backup. Choose Merge (add to existing) or Replace (start fresh from file).

Print Report — Generate a filtered, human-readable HTML report you can print or share. Filter by animal, log type, and date range.

👥 Multiple Users

Go to Settings → Active User. Add each farm hand's name. The active user is stamped on every log so you know who recorded what.

Change the active user any time — just tap their name in the dropdown.

🔄 farmOS Sync

If you have a farmOS server, enter your URL and credentials in Settings. All logs are saved locally first, then pushed to farmOS when you sync.

Push to farmOS — Sends all unsynced logs to your server.

Fetch Inventory — Pulls your animal list and stock items from farmOS.

No server? Enable Standalone Mode in Settings and use XML export/import instead.

⚙️ Settings Tips

Farm Name — Used in exported filenames. Set this first.

Modules — Turn Goats or Stock on/off to simplify the interface for your farm.

Farm Logo — Pick a photo to show in the header.

Clear All Logs — Wipes all logs and stock inventory. Use before switching to a different farm.
